Night Rocket Camera

This project was born out of a desire to capture a picture of my night rocket in flight. Originally the idea was to trigger it on launch and take a long exposure photo. However, after getting the parts for the project, I found a tequinique to fake a long exposure photo using a video and some python code.

This site allows photos and videos to be taken using the camera attached to the Raspberry Pi that the site is served off of. Videos can be rendered into long exposure photos using the built in long exposure tool. Because the long exposure photos are rendered from viedeos, the start and end points can be changed at will, making this approach very flexible.

To take a picture using the camera, go to Camera Settings. To view pictures and videos that have already been taken, go to Gallery.

This page uses Tornado to run the webserver. The code for this project can be found on GitHub.
